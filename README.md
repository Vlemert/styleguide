# Pencil Styleguide

## Installation

Run NPM setup

  ```
  npm run setup
  ```

Or

  ``` 
  npm install
  gem install jekyll 
  ```

## Watch Sass changes and build docs

  ```
  npm run watch
  ```
  
## Build once

  ```
  npm run build
  ```

## Dependencies

- [GulpJS](http://gulpjs.com)
- [scss-wipe](https://github.com/marsbergen/scss-wipe)
