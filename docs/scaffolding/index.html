HTML5 Doctype

Craft makes use of certain HTML elements and CSS properties that require the use of the HTML5 doctype. Include it at the beginning of all your pages

<!DOCTYPE html>
<html lang="en">
  ...
</html>

Box-sizing

We reset box-sizing to border-box for every element in Craft. This allows us to more easily assign widths to elements that also have padding and borders.

Build on (S)CSS-Wipe

For improved cross-browser rendering, we use SCSS-Wipe, which is a SCSS port for CSS-Wipe.

Some features of CSS-Wipe are:

  • removes all padding, margin and border
  • wipes decorations
  • sets box-sizing: border-box for predictable size calculations
  • resets list style types
  • reset styles on bleeding edge webkit elements
