How to develop the Style guide

Install it on your machine

Run NPM setup

Hint: There is no need to run npm install manually

npm run setup

Or

gem install bundler
bundle install
npm install

Watch Sass changes and build docs (dev style)

npm run watch

Build the CSS and Docs just once (production style)

npm run build

Build upon

  • Craft
  • GulpJS
  • scss-wipe
